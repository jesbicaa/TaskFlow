# TaskFlow - Gestão Ágil

Sistema web básico de gerenciamento de tarefas desenvolvido com Flask, focado na aplicação de Engenharia de Software e metodologias ágeis (Kanban).

## Como rodar
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o sistema: `python src/app.py`
3. Acesse no navegador: `http://127.0.0.1:5000`
